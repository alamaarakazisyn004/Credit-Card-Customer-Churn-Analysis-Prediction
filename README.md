## Project Overview

This project focuses on analyzing customer behavior and predicting churn using a credit card dataset. It includes data preprocessing, feature engineering, machine learning models, and dashboard visualization.

## Objectives
Clean and preprocess raw data
Perform exploratory data analysis
Build regression and classification models
Predict customer churn
Create business insights using Tableau

## Tools & Technologies
Python (Pandas, NumPy, Scikit-learn)
Data Visualization (Matplotlib, Seaborn)
Tableau

## Dataset
Credit Card Customer Attrition Dataset

## Steps Performed
Data Cleaning and Preprocessing
Feature Engineering
Model Building (Regression & Classification)
Model Evaluation
Dashboard Creation

## Conclusion
The project successfully identifies high-risk customers and provides insights to improve customer retention strategies.
